# dotfiles

Personal configuration files for Hyprland + Zsh.

## Contents

- `hypr/` — Hyprland window manager config
- `zshrc/` — Zsh customization files (loaded by ML4W zshrc)
- `.p10k.zsh` — Powerlevel10k prompt config

## Install

```bash
git clone https://github.com/mhpsy/dotfiles.git ~/dotfiles
cd ~/dotfiles
chmod +x install.sh
./install.sh
```

## Notes

- Zsh config uses [ML4W](https://github.com/mylinuxforwork/dotfiles) as the base loader. Install ML4W first, then run this install script.
- Hyprland config also based on ML4W dotfiles.
